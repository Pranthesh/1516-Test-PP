6. Cookies and Third-party Applications Cookies

  • Maxim’s uses cookies on this Website. A cookie is a small text file that is placed on your hard disk by a Web page server. A cookie is not designed to access any information (except for the information of the cookie itself) on the user’s computer. On the contrary, it is an identifier that can later be searched by the same Web page server to identify a particular computer.
  • This Website can link cookies to other information such as personal identifiers, webpages visited on this Website, IP address, previous websites visited, types of browsers and operating systems used, and the length of time spent on this Website.
  • Similarly, Maxim’s may use third-party applications on this Website.
  • The purposes of using cookies or third-party applications are to enable automated sign-in and registration so that you are linked to information for online purchase (if any), to adjust this Website according to your preferences, to personalize promotions or marketing, to analyze the usage of different webpages, and to develop and enhance this Website.
  • You can modify your web browser settings regarding cookies to send you warnings or to decline acceptance of cookies. Please note that this Website may not function normally or permit your sign-in if you decline acceptance of cookies. For further details about our use of cookies, please refer to our Cookie Policy.

7. Use of Web Beacons

  • Maxim’s may use Web beacons on this Website or other websites and allow third parties to place Web beacons on this Website to monitor the effectiveness of promotions or for other legal purposes.
  • Web beacons, also called web crawlers, are objects embedded in a web page or email which allow a website owner or a third party to check whether a user has viewed the page or email. They are usually invisible to the user as their size is generally very small (1x1 gif) and have the same colour as the web page, document, or email. Web beacons are identified as HTML IMG tags on web pages.
  • Web beacons can be linked to personal information. For example, advertising networks may use web beacons to associate the web pages visited with the personal information of users and, based on such information, add particular kinds of advertising bars. Web beacons can also independently track the number of people viewing the website.
  • Users can click on the information option on the web page to find out whether Web beacons are used on the page.
